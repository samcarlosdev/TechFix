# TechFix - Sistema de Gerenciamento de Assistência Técnica

> **Status do Projeto:** ⚠️ Em Desenvolvimento (PROJETO INTEGRADOR)

## 🎯 Objetivo do Software
O TechFix foi desenvolvido para centralizar o controle de uma assistência técnica de computadores, permitindo o registro de clientes, a gestão de ordens de serviço por diferentes níveis de usuários e a emissão de recibos em PDF.

## 🚀 Tecnologias Aplicadas
* **Linguagem:** Java
* **Interface Gráfica:** Java Swing (NetBeans IDE)
* **Gerenciador de Dependências:** Maven
* **Persistência de Dados:** JPA (Java Persistence API) / Hibernate
* **Banco de Dados:** MySQL
* **Relatórios:** iText PDF

## ✅ Funcionalidades (Requisitos)
1. **Controle de Acesso:** Login diferenciado para Técnicos e Assistentes.
2. **Gestão de Clientes:** Cadastro e edição de dados de contato.
3. **Registro de Serviços:** Cadastro detalhado de manutenções realizadas.
4. **Histórico:** Consulta de serviços anteriores com filtros.
5. **Edição e Exclusão:** Gestão completa de serviços (CRUD).
6. **Emissão de Recibos:** Geração de arquivo PDF com dados do serviço.
